# discord-gpt

Chat with a Discord bot powered by GPT.

## Setup

1. Install requirements
```python
pip install -r requirements.txt
```
2. Create a .env file
```shell
touch .env
```
3. Populate the .env with the bot's Discord token
```python
DISCORD_TOKEN=<token>
```
4. Start the bot
```python
python3 bot.py
```
